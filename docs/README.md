## 项目概览

```
Data Analysis Agent
├── 前端 (Next.js 15)
│   ├── App Router 页面路由
│   ├── React 组件库
│   ├── Zustand 状态管理
│   └── TailwindCSS 样式
│
├── 后端 (Python FastAPI)
│   ├── Excel 数据处理
│   ├── AI Agent (LangChain)
│   ├── 报告生成 (Word/PPT)
│   └── RESTful API
│
└── 数据库 (Prisma + SQLite/PostgreSQL)
    ├── 用户认证
    ├── 文件管理
    └── 分析结果存储
```

## 技术栈摘要

| 层级 | 技术 |
|------|------|
| **前端框架** | Next.js 15.1.4, React 19 |
| **UI组件** | Radix UI, TailwindCSS 3.4 |
| **状态管理** | Zustand 5.0 |
| **后端框架** | FastAPI 0.115, Python 3.10+ |
| **AI框架** | LangChain 0.3, OpenAI/Qwen |
| **数据库** | SQLite (开发) / PostgreSQL (生产) |
| **ORM** | Prisma 6.2 |
| **容器化** | Docker, Docker Compose |
